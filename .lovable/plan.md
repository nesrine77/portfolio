## Design Features

### Visual Design
- Primary color (#d95448) for buttons, links, accents
- Background (#f2e4e4) for section backgrounds
- Clean typography with clear hierarchy
- Professional, modern aesthetic

### Dark/Light Mode
- Smooth transition between modes
- Persistent preference (saved to localStorage)
- Carefully designed color schemes for both modes

### Animations & Interactions
- Smooth scroll navigation
- Fade-in animations on scroll
- Hover effects on buttons, cards, and links
- Loading states for images
- Subtle parallax effects

### Responsive Design
- Fully responsive across desktop, tablet, and mobile
- Mobile-optimized navigation
- Flexible grid layouts
- Touch-friendly interactions

### Performance & Accessibility
- Optimized images (Nesrine's photo and logo)
- Semantic HTML structure
- ARIA labels for accessibility
- SEO-friendly meta tags
- Fast loading with lazy loading for images
- Error handling for failed image/link loads
